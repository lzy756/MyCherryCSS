2️⃣ **CherryYou**

   🖼️ 文件名: `CyberCard.css`

   🖼️ 文件名: `cyberline.css`

🔧 **安装指南**

1️⃣ **获取主题文件**
   📥 在仓库中选择想要的主题
   📋 点击 "Raw" 按钮复制内容

2️⃣ **应用主题**
   ⚙️ 打开 CherryStudio
   🔍 进入 `设置` > `显示设置` > `自定义 CSS`
   📝 粘贴代码到 CSS 编辑区

3️⃣ **完成设置**
   🔄 重启应用以确保样式正确加载
   🎉 开始享受全新的视觉体验！
