## 🚀 使用指南

1️⃣ **获取主题**
📥 从仓库下载所需主题文件
📋 点击 "Raw" 按钮复制内容

2️⃣ **应用主题**
⚙️ 打开 CherryStudio
🔧 导航至: 设置 > 显示设置 > 自定义 CSS
📝 粘贴复制的 CSS 代码

3️⃣ **开始体验**
✅ 主题将立即生效
🔄 建议重启应用以确保样式完全加载
